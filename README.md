# ❌⭕ Tic Tac Toe

🔗 **Live Demo:** https://sksingh04.github.io/Tic-Tac-Toe---Suraj-Singh/

A responsive Tic Tac Toe game built using **HTML, CSS, and JavaScript** where two players can compete in a classic 3×3 board.

---

## 🎮 Features

- 👥 Two Player Mode
- 🏆 Winner Detection
- 🤝 Draw Detection
- 🔄 Restart Game
- 📱 Responsive Design
- 🎨 Clean Modern Interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
